🏗️ Arquitetura Implementada:

Cliente (Frontend):

  • Usa apenas pk_test_ (seguro para exposição)
  • Cria requisição para servidor
  • Redireciona para Stripe Checkout

Servidor (Backend):

  • Usa sk_test_ (privada, segura)
  • Cria Checkout Sessions
  • Retorna URL de redirecionamento
